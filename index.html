What is this Project about?
This Project is about working with APIs. We have used 3 APIs:
  • Google Maps API for displaying maps (free version came with water mark - For development Purpose only)
  • OpenCageData API for Geocoding to find region code using latitude and longitude co-ordinates
  • YouTube Data V3 API for searching top 10 trending videos for that region as per the categories i.e. Music, News, Sports, Gaming and Movies
    • Instructions
    • By default, when you land on the website map is centered to Toronto and a marker is placed. You will be able to see trending videos of this region.
    • If you want to switch the genre of trending videos, simply select your desired category on the top right corner.
    • To change the region move around the map and place the marker on any country. You will see name of the country below the map and trending videos of that region in the YouTube section.
    • Try out the website by clicking the mockup image on the right!
